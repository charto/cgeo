cgeo
====

Collection of geometry data types:

- Point
- LineString
- Polygon
- MultiPoint
- MultiLineString
- MultiPolygon
- GeometryCollection
- CircularString
- CompoundCurve
- CurvePolygon
- MultiCurve
- MultiSurface
